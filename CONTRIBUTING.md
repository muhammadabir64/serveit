# Contributing to ServeIt

Thanks for your interest in contributing.

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install dependencies:

```bash
npm install
```

4. Start development:

```bash
npm run dev
```

## Development Rules

- Keep changes focused and small
- Follow existing code style and project structure
- Test your changes manually before opening a PR
- Run lint before pushing:

```bash
npm run lint
```

## Pull Request Process

1. Create a branch from `main`
2. Commit with clear messages
3. Push branch and open a Pull Request
4. Describe what changed and how to test it
5. Link related issues (if any)

## Reporting Bugs / Requesting Features

- Use GitHub Issues
- Provide clear steps to reproduce (for bugs)
- Include screenshots/logs when possible
